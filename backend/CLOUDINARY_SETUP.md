# ===============================================

# CLOUDINARY SETUP GUIDE

# ===============================================

#

# 1. Tạo tài khoản miễn phí tại: https://cloudinary.com/

#

# 2. Sau khi đăng ký, vào Dashboard:

# https://console.cloudinary.com/console

#

# 3. Copy các thông tin sau:

# - Cloud Name (ví dụ: dxy123abc)

# - API Key (ví dụ: 123456789012345)

# - API Secret (ví dụ: AbCdEfGhIjKlMnOpQrStUvWxYz)

#

# 4. Paste vào file .env (không commit file này lên Git!)

#

# ===============================================

# Cloudinary Configuration

CLOUDINARY_CLOUD_NAME=your-cloud-name
CLOUDINARY_API_KEY=your-api-key
CLOUDINARY_API_SECRET=your-api-secret
